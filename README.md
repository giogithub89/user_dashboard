### Files explaination

`customFormDialog.jsx` => src/components/modalDialog/customFormDialog.jsx, It's a custom dialog to edit or to add new interests, categories and bags.

`useData.js` => src/api/useData.js , it's a custom general API to fetch, post data as in Reset Password, Edit password from staff profile page. `deleteCategory` => it's a custom general API to delete categories and bags.

`imageUpload.js` is to upload single image on staff profile or users profile using bbase64, while `multipleFileUpload.js` is for single and multiple image upload with formData, for the content.
